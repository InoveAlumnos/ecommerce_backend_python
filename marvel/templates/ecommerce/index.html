¿Cómo obtengo mi Api-Key?


1. Registrate

Opcion a: Hacerlo vía Página Web

Para registrarse tenés que ir a la página de registro y crear tu usuario.


Opcion b: Hacerlo vía Request

Para registrarse tenés que enviar un request de tipo POST a https://inove-marvel-backend.herokuapp.com/ecommerce/clients/signup y crear tu usuario, para ello, tu request debe ser de la siguiente forma


Headers 
    
        {
            "Content-Type": "application/json"
        }

Body 
    
        {
            "first_name": "tu nombre",
            "last_name": "tu apellido",
            "email": "tu email",
            "username": "acá elegís tu username",
            "password": "acá elegís tu password"
        }



2. Iniciar Sesión

Opción a: Hacerlo vía Página Web

Para iniciar sesión tenés que ir a la página de inicio de sesión e indicar tus credenciales.


Opción b: Hacerlo vía Request

Para iniciar sesión tenés que enviar un request de tipo POST a https://inove-marvel-backend.herokuapp.com/ecommerce/clients/login indicando tus credenciales. para ello, tu request debe ser de la siguiente forma


Headers 
    
        {
            "Content-Type": "application/json"
        }

Body 
    
        {
            "username": "tu nombre de usuario",
            "password": "tu contraseña"
        }
